This circuit consists of an ultrasonic sensor and an LED strip connected through an n-mosfet transistor. Depending on the distance detected by the ultrasonic sensor, the brightness of the LED strip changes, with closer distance resulting in a dimmer light.

The LED strip I am using says it can handle up to 12V/1.5A of power, so seeing that the power supply is providing less than that, I think the LED strip will use all that is given but being a bit dimmer than if it was receiving as much as it could take. Since the maximum current for the diode body is 80A of current, I think the circuit is safe since I am only using 1A of current, which is much lower than 80A.

Circuit's Operation gif

This circuit uses an ultrasonic sensor and a library to read the distance detected by the sensor. Depending on the distance, from 5, 10, 15 cm or more, the brightness of the LED strip changes from incriments of 50. The closer the object is to the sensor, the dimmer the light appears.

Code Snippet

        
        // library for the ultrasonic sensor
        #include Ultrasonic.h 
        // initializes ultrasonic and its echo and trig pins
        Ultrasonic ultrasonic(3, 2);
        // initialises integer for holding distance
        int distance;
        // initialises integer for changing the brightness
        int brightness;

        void setup() {
          // sets pin 5 as an output, its current state would 
          // not be receiving anything yet until it is called 
          // later during the loop()
          pinMode(5,OUTPUT);
        }

        void loop() {
          // sets the distance integer as the value received from
          // the ultrasonic sensor using the library's read property
          distance = ultrasonic.read();
          // changes the brightness of the LED strip depending on the 
          // distance received from the ultrasonic sensor
          if (distance <= 5) {
            // sets the brightness to 50 if conditions are met
            brightness = 50;
            // sends info to pin 5 and changes to adjusted brightness
            analogWrite(5,brightness);
          } else if (distance <= 10) {
            brightness = 100;
            analogWrite(5,brightness);
          } else if (distance <= 15) {
            brightness = 150;
            analogWrite(5,brightness);
          } else {
            brightness = 200;
            analogWrite(5,brightness);
          }
        }
