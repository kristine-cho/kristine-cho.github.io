I used 3 LEDs, 3 resistors, and an Arduino to create a blinking circuit. Each LED is connected to a different pin on the Arduino so that they can blink independently. My circuit runs with a 0.5 second timer, turning a light on for 0.5 seconds and off for 0.5 seconds and goes through each lightbulb in rotation.

Code Snippet

        
        // initializing a timer for 1 second
        int timer = 500; 

        // the setup function runs once when you press reset or power the board
        void setup() {
          // for loop initializing pins 10-12 as an output
          for (int thisPin = 10; thisPin < 13; thisPin++){
            // initializing thisPin as an output
            pinMode(thisPin, OUTPUT);
          }
        }

        // the loop function runs over and over again forever
        void loop() {
          // forloop for turning each pin on and off from pin 10-12
          for (int thisPin = 10; thisPin < 13; thisPin++) {
            //turns on the LED by making the voltage high
            digitalWrite(thisPin, HIGH);
            //waits 1 second
            delay(timer);
            //turns off the LED by making the voltage low
            digitalWrite(thisPin, LOW);
            //waits 1 second
            delay(timer);
          }
        }
